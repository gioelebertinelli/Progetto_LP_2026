# Progetto_LP_2026
Implementazione nei linguaggi Prolog e Lisp di un compilatore da espressioni regolari a
NFSA usando l'algoritmo di Thompson.
Il programma permette di verificare se un termine rappresenta una regex valida, compilare un'espressione regolare in un automa NFSA salvandone la struttura e riconoscere se una stringa appartiene al linguaggio definito dall'automa.
